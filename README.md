# Sistema-Restaurante

## Documentación Swagger

Al iniciar el servidor, la documentación interactiva queda disponible en:

- `http://localhost:PUERTO/Restaurante/v1/docs`

También puedes obtener el spec OpenAPI en formato JSON en:

- `http://localhost:PUERTO/Restaurante/v1/docs.json`

> Reemplaza `PUERTO` por el valor definido en tu archivo `.env` (`PORT`).
